在职业生涯中，他在普渡大学计算机科学系担任高级系统程序员和工程师超过30年，是师生们宝贵的资源。他还是软件公司Rnet的联合创始人，职业生涯早期曾在宝来公司从事大型机操作系统工作。

除了技术工作，戈布尔还被铭记为一位充满热情的巧手匠、业余无线电操作员（呼号K9TRV）以及在当地乐队演奏的音乐家。他是一位深爱家庭的人，身后留下了妻子、子女和孙辈。他的遗产是一位才华横溢、富有创造力且乐于助人的问题解决者，在计算历史和互联网文化上都留下了印记。

## 10. JSON画布规范

**原文标题**: JSON Canvas Spec

**原文链接**: [https://jsoncanvas.org/spec/1.0/](https://jsoncanvas.org/spec/1.0/)

JSON Canvas 规范（版本 1.0）定义了一种使用 JSON 表示交互式、基于节点的画布的格式。画布围绕两个主要数组构建：`nodes`（视觉对象）和 `edges`（连接线）。

**节点**是主要元素，分为四种类型：
*   **文本：** 包含 Markdown 格式的文本。
*   **文件：** 引用文件路径，可选地包含子路径（例如 `#heading`）。
*   **链接：** 包含一个 URL。
*   **组：** 作为其他节点的视觉容器，可以具有标签和背景图像。

所有节点共享必需的属性：唯一的 `id`、`type`、基于像素的 `x`、`y`、`width`、`height`，以及可选的 `color`。节点按照它们在数组中出现的顺序进行渲染，这决定了它们的视觉堆叠顺序（z-index）。

**边**用于连接节点。每条边都有一个唯一的 `id`、一个 `fromNode` 和一个 `toNode`（引用节点 ID），以及用于控制连接点（`fromSide`/`toSide`）、端点形状（`fromEnd`/`toEnd`）、`label` 和 `color` 的可选属性。

**颜色**可以指定为十六进制代码（例如 `"#FF0000"`），或使用六种编号预设颜色（`"1"` 到 `"6"`）中的一种，这些预设颜色对应红色、橙色、黄色、绿色、青色和紫色。预设颜色的具体值由应用程序定义。

## 12. 雨果的新CSS超能力

**原文标题**: Hugo's New CSS Powers

**原文链接**: [https://www.brycewray.com/posts/2026/04/hugos-new-css-powers/](https://www.brycewray.com/posts/2026/04/hugos-new-css-powers/)

本文探讨了Hugo的`css.Build`函数在管理网站样式方面的能力与局限。文章重点指出了三大优势：**打包**多个CSS文件、为生产环境**压缩**代码，以及通过底层的`esbuild`工具进行**后处理**（例如将现代CSS转译为旧版浏览器可识别的代码）。

然而，作者也指出了显著的**局限性**。后处理功能受限于`esbuild`所支持的特性，可能无法满足所有现代CSS需求。开发者必须查阅`esbuild`文档并进行充分测试。

文章随后将`css.Build`与其他处理器进行了比较：
*   **Sass**：提供嵌套和混合等强大功能，但属于预处理器，需要额外工具实现浏览器兼容。
*   **PostCSS**：通过插件高度可配置，但基于JavaScript且速度较慢。
*   **Lightning CSS**：速度极快，但需要与Hugo进行复杂集成（“强行适配”）。

结论是，`css.Build`是一种快速便捷的解决方案，适用于面向现代浏览器（如Baseline 2023/2024）的网站。若项目需要更广泛的浏览器支持或高级CSS功能，开发者可能需要结合其他处理器使用，并接受由此增加的复杂性。

## 27. 《叶甫盖尼·奥涅金》文本的统计研究示例——马尔可夫，1913年 [pdf]

**原文链接**: [https://alpha60.de/research/markov/DavidLink_AnExampleOfStatistical_MarkovTrans_2007.pdf](https://alpha60.de/research/markov/DavidLink_AnExampleOfStatistical_MarkovTrans_2007.pdf)

这份PDF是安德烈·马尔可夫1913年论文《叶甫盖尼·奥涅金文本统计研究示例》的数字化版本。该文章是概率论与数理语言学领域的奠基性著作。

马尔可夫的核心目标是通过具体文学案例验证其链式理论（即今所称的马尔可夫链）。他以亚历山大·普希金的诗体小说《叶甫盖尼·奥涅金》为对象，系统分析了元音与辅音的序列规律。

其主要研究步骤包括：
1. 选取文本前20,000个字母
2. 将每个字母严格划分为元音或辅音
3. 系统统计两类状态间的转移情况（如元音转辅音、辅音转元音、元音转元音）

基于这些数据，马尔可夫计算了状态转移概率。其核心发现是：下一个字母为元音的概率，显著受当前字母是元音还是辅音的影响。这证明字母序列并非独立存在，而是构成了每个事件的概率受前一事件影响的“链式结构”。

本研究具有重要历史意义，它是随机过程理论首次应用于非数值型现实数据集的范例之一。该研究突破了概率论对独立试验的传统局限，为马尔可夫链的发展奠定基础，后者后来成为计算机科学、统计学与自然语言处理等领域的核心工具。
