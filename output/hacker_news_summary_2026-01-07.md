## 3. Shipmap.org

**原文标题**: Shipmap.org

**原文链接**: [https://www.shipmap.org/](https://www.shipmap.org/)

Shipmap.org是由Kiln与伦敦大学学院能源研究所合作创建的2012年全球货船动态交互式可视化地图。该地图以海底地形为底图，展示商船航线与实时位置，支持追踪五类船舶：集装箱船、干散货船、油轮、气体散货船和车辆运输船。图中实时显示二氧化碳排放量（以千吨计）与货运能力统计。

用户可通过平移、缩放和时间轴导航查看船舶动态，并可按船舶类型筛选或切换港口、航线等图层。该项目由欧洲气候基金会资助，采用exactEarth的AIS数据与克拉克森的船舶资料，伦敦大学学院能源研究所基于这些数据计算了二氧化碳排放量。

创建者指出数据存在一定局限：2012年初数据覆盖不全，且因数据缺失偶尔出现船舶看似穿越陆地的视觉误差。网站提供高分辨率印刷品销售，并欢迎标注来源的嵌入使用。该工具旨在展现全球海运贸易的规模及其环境影响。

## 9. 在macOS / Linux / Windows上通过FUSE使用原生Amiga文件系统

**原文标题**: Native Amiga Filesystems on macOS / Linux / Windows with FUSE

**原文链接**: [https://github.com/reinauer/amifuse](https://github.com/reinauer/amifuse)

**amifuse** 是一款工具，允许你在现代 macOS、Linux 和 Windows 系统上挂载 Amiga 硬盘和软盘镜像（如 HDF、ADF）。其核心创新在于通过模拟 m68k CPU 使用**原生 Amiga 文件系统驱动程序**（如 PFS3、SFS），确保高兼容性，无需依赖逆向工程代码。

**主要特性：**
*   **跨平台支持：** 通过 FUSE（macFUSE、Linux FUSE、WinFSP）运行。
*   **驱动程序支持：** 可使用磁盘镜像 RDB（Rigid Disk Block）中嵌入的驱动程序或手动指定。
*   **镜像格式：** 支持 HDF/RDB 镜像、Emu68 风格的 MBR 镜像以及 ADF 软盘镜像。
*   **使用方式：** 通过简单命令挂载镜像（`amifuse mount disk.hdf`）。在 Linux 上需要明确指定挂载点。
*   **工具集：** 包含 `rdb-inspect` 用于检查分区和提取嵌入的驱动程序，以及 `driver-info` 用于检查处理程序二进制文件。
*   **实验性功能：** 包括读写模式以及仅在 macOS 上将 Amiga `.info` 图标转换为原生 Finder 图标的功能。

**快速开始：** 安装依赖项并克隆仓库后，可通过 `make download` 和 `make unpack` 下载测试镜像和处理程序，然后直接挂载。工具在前台运行；按 Ctrl+C 卸载。
