MapLibre 推出了 **MapLibre Tile (MLT)**，这是一种新的矢量切片格式，旨在作为广泛使用的 Mapbox Vector Tile (MVT) 的继任者。MLT 从头开始构建，旨在应对现代挑战，包括不断增长的地理空间数据量和复杂的新数据源，同时充分利用当代硬件和图形 API。

该格式目前的主要优势是**显著改进的压缩率**（对于大型切片最高可达 6 倍）和**更快的解码性能**，这是通过面向列的布局和轻量级编码实现的。这减少了延迟和存储成本，并提高了缓存利用率。

MLT 的设计也考虑了未来的功能，包括更好地支持 3D 坐标/高程、适用于 Overture Maps 等格式的线性参考、复杂数据类型（嵌套属性、列表），以及针对高效 GPU 和 CPU 处理优化的格式。

对于即时使用，MapLibre GL JS 和 MapLibre Native 现在都通过 `"encoding": "mlt"` 属性支持 MLT 数据源。开发者可以使用演示样式、转换服务器或即将推出的 Planetiler 切片生成器进行实验。

该项目是多年合作的成果，并通过 Slack 和 GitHub 开放给社区进行反馈和贡献。

**原文标题**: Dithering – Part 2: The Ordered Dithering

**原文链接**: [https://visualrambling.space/dithering-part-2/](https://visualrambling.space/dithering-part-2/)

本文是抖动技术系列的第二部分，重点介绍**有序抖动**——一种将灰度图像转换为黑白图像的技术。

文章首先回顾了量化（减少色彩层次）的概念，并解释了单一阈值的不足。有序抖动则使用**阈值图**，即由多个阈值组成的网格，平铺于整幅图像上。每个像素的灰度值与图中对应位置的阈值进行比较，从而决定其变为黑色或白色，由此形成的图案能够模拟出灰色调。

文章指出了一个关键问题：排列不当的阈值图会产生视觉伪影。为此引入了**拜耳矩阵**作为解决方案，该矩阵通过特定方式排列阈值，生成交叉阴影图案，使黑白像素均匀分布。从2x2矩阵（4种灰度）开始，文章展示了将矩阵尺寸增大至4x4（16种灰度）和8x8（64种灰度）如何创建更平滑的渐变并减少生硬的过渡。

最后，文章简要探讨了拜耳系列之外的其他阈值图排列方式，例如**簇点矩阵**（可产生经典报纸印刷效果）和**空隙与集群法**（能生成更自然、更少规则感的“蓝噪声”图案）。文章结尾预告了下一个主题：误差扩散抖动。

**原文标题**: There is an AI code review bubble

**原文链接**: [https://www.greptile.com/blog/ai-code-review-bubble](https://www.greptile.com/blog/ai-code-review-bubble)

在这篇文章中，AI代码审查工具Greptile的创始人认为，市场正经历着同类产品的泡沫。他们指出，仅凭主观声称性能更优来竞争是无效的。

相反，他们通过基于三大核心原则的长期愿景来区分自己的公司：

1.  **独立性：** 他们认为审查代码的AI必须与编写代码的AI分离。让同一个代理既生成又批准代码被视为根本性的冲突，类似于学生给自己的论文打分。

2.  **自主性：** 他们预测代码验证（审查、测试、质量保证）将完全自动化——这是人类不喜欢且不擅长的工作。因此，Greptile被构建为一个用于后台自动化的“管道”产品，没有人工审查界面，为未来极少人工干预做准备。

3.  **反馈循环：** 他们设想一个无缝的循环：人类提供意图，编码代理编写代码，审查代理验证代码。这形成了一个自动化循环，问题被退回修复，直到代码自动批准并合并，仅在需要澄清时才咨询人类。

文章总结道，选择代码审查工具是一个重大的长期决策。虽然未来充满不确定性，但Greptile致力于其自主、独立的代码验证愿景。

## 10. 称量地球的山

**原文标题**: The mountain that weighed the Earth

**原文链接**: [https://signoregalilei.com/2026/01/18/the-mountain-that-weighed-the-earth/](https://signoregalilei.com/2026/01/18/the-mountain-that-weighed-the-earth/)

这篇文章解释了科学家如何首次利用一座山测量地球的质量。牛顿定律表明，地球对一个人的“重量”等于这个人在地球上的重量，但这本身并未揭示地球的质量。要计算地球质量，科学家需要测量一个已知较小物体的引力。

1774年，皇家天文学家内维尔·马斯基林在苏格兰领导了希哈良山实验。通过测量山脉引力导致的铅垂线微小偏转（仅0.0032度），并结合查尔斯·赫顿对山脉形状和岩石密度的详细勘测，他们得以比较引力大小。

由此，他们计算出地球的平均密度，进而得出其质量。他们得出的结果4.87×10²¹公吨与现代值（5.91×10²¹公吨）误差在20%以内。尽管亨利·卡文迪许1798年的扭秤实验后来实现了更高精度，但希哈良山实验具有开创性意义。它首次证明引力是源自所有物质（不仅是天体）的普遍作用力，并成功完成了人类对地球质量的第一次测量。
