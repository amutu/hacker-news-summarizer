Flipper One是一款全新的开源Linux掌上电脑项目，与Flipper Zero不同。它专注于IP网络（第一层及以上），而Zero处理离线协议（第零层）。核心目标包括打造全球最开放的ARM计算机，支持完整主线Linux内核，消除二进制固件，并采用协处理器架构（RK3576 CPU + RP2350 MCU）。MCU可在Linux关闭时独立运行，控制显示与电源。

## 5. Show HN：Freenet，一个用于去中心化应用的点对点平台

**原文标题**: Show HN: Freenet, a peer-to-peer platform for decentralized apps

**原文链接**: [https://freenet.org/](https://freenet.org/)

**Freenet 简介**

Freenet 是一个为去中心化应用（dApps）设计的点对点平台，支持通信、协作和商业活动，无需依赖大型科技基础设施。其核心技术采用“小世界”环形拓扑结构组织网络节点，信息仅需数跳即可高效传输，无需服务器即可扩展至数百万用户。

**主要功能与受众：**

- **用户：** 应用在标准浏览器中运行，功能类似普通网站，但具备抗审查（无法被屏蔽）、隐私优先（无追踪）特性，且以点对点方式运作，而非依赖云服务器。
- **开发者：** 可使用 Rust 和 TypeScript 等常用工具构建应用，并直接部署至全球 Freenet 网络——无需服务器维护、云成本及服务条款限制。
- **支持者：** 该项目由一支小型团队通过资助和捐赠开发，并邀请各界共同建设去中心化互联网基础设施。

更多资源包括实时网络仪表盘、30分钟技术讲座、视频演示、播客、Matrix 聊天室、GitHub 仓库及常见问题解答。项目强调开放性、互操作性和不可阻挡的应用。

## 6. 在2021款MacBook上通过Gemma4-31B本地索引一整年视频（50GB交换空间）

**原文标题**: Indexing a year of video locally on a 2021 MacBook with Gemma4-31B (50GB swap)

**原文链接**: [https://blog.simbastack.com/indexed-a-year-of-video-locally/](https://blog.simbastack.com/indexed-a-year-of-video-locally/)

作者介绍如何在一台2021款MacBook Pro上搭建本地AI视频索引系统，以解决马赛马拉未标注旅行素材积压的问题。在放弃云端AI剪辑工具（成本过高、生成式AI不适合真实旅行品牌、且所有工具均假设素材已标注）后，作者意识到核心问题并非剪辑而是索引——让素材能用自然语言搜索。

他们构建了一套基于侧挂文件的本地处理流程：每个视频片段经过ffprobe、exiftool、ffmpeg（帧提取）、WhisperX（转录）、insightface（人脸嵌入）和视觉大语言模型（通过LM Studio本地运行的Gemma 4 31B，占用28GB内存+50GB交换空间）分析。输出结果为每个片段生成包含YAML元数据（光照、人数、关键词、GPS、质量评级）和散文体描述的`.description.md`文件。

核心经验：枚举约束比开放式提示更能防止模型幻觉；本地31B模型在批量索引任务中已接近云端水平；AI视频编辑器解决的是错误问题——索引才是前置条件。下一步将构建编辑器层（Claude Code + DaVinci Resolve MCP）。作者指出，这台2021年M1 Max本是为运行Chrome而购入，如今却运行着设计时未曾预料的310亿参数模型，在周末批量处理中承受50GB交换空间而未受损。

**《英国红色电话亭都去哪儿了？》摘要**

本文介绍了“K6项目”——一项记录英国现存所有红色电话亭位置的个人计划。该项目旨在追踪这些因手机普及而退役的标志性建筑，揭示它们的新用途与社区故事。

项目已建立包含**6745个已考察电话亭**的详细数据库，其中大部分（**5177个**）位于英格兰，苏格兰（**855个**）和威尔士（**713个**）紧随其后。主要分类包括：

- **电话亭类型**：经典**K6**型号占主导（6588个），另有K8及其他稀有型号。
- **用途**：最常见用途为**“保留电话”**（1922个）、**除颤仪**（1406个）、**图书交换站**（1208个）及**“闲置”**（1405个）。其他改造包括艺术空间、博物馆、食品银行和种子银行。

- **其他数据**：多数电话亭为红色（6426个），带有都铎王冠标识（4720个）。**1796个**被列为保护建筑。

该项目持续更新，近期报道涉及设得兰群岛、马里郡及北安普敦郡的电话亭。它如同一幅完整地图，展示了这些红色亭子的现存分布及社区赋予它们的新生。

## 9. 我的4.8万美元GPU服务器值吗？

**原文标题**: Was my $48K GPU server worth it?

**原文链接**: [https://rosmine.ai/2026/05/13/was-my-48k-gpu-worth-it/](https://rosmine.ai/2026/05/13/was-my-48k-gpu-worth-it/)

2024年从FAANG离职的独立研究员Rosmine打造了一台名为"grumbl"的GPU服务器，配备6块RTX 6000 Ada显卡，总成本4.8万美元。他选择这些显卡而非A100（不支持FP8）和H100，是因为其性价比更高。由于公寓电力限制，服务器需在两个独立电路上连接两个电源，他因此聘请了专业建造师。后来这台服务器迁至他父母家的地下室。

他将自建服务器与租用云GPU进行了比较。通过详细日志记录，他发现平均GPU利用率为76%（2025年1月以来达85%），低于预期。同等云租赁成本需6.8万美元，因此他节省了1.7万美元，目前每天节省90-105美元。电费约3000美元。

然而，真正目标是支持高风险实验。他最终解决了一个重大LLM问题，成功发布后获得超过40万次浏览和多家公司关注。关键建议：自建服务器可能遭遇GPU互联速度慢的问题；租赁会让实验显得成本高昂，而自建则让闲置GPU显得浪费。他还指出维护耗时，下次会考虑托管中心。
