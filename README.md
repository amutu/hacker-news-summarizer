**Wyrm** 是一个用纯 TypeScript 编写的开源（MIT 许可证）符号代数引擎，专为构建基于触摸的数学界面而设计，用户可通过拖拽项或展开幂等手势来解方程。

**核心原则：** 方程从不被验证，仅通过合法强制执行的改写规则进行变换。每个可达状态**在构造上是有条件正确的**。可能导致丢根（例如，除以 \(b\) 要求 \(b \neq 0\)）或增根（等式两边平方）的操作是允许的，但其条件会变为可见的、一等公民的**假设**，随方程一同传递。

**主要特性：**
- **精确算术：** 仅使用大整数有理数，无浮点数。
- **不可变抽象语法树（AST）：** 具有稳定的节点 ID，支持基于 ID 的动画。
- **移动枚举：** 根据当前假设返回所有合法手势选项。
- **约 25 条内置规则：** 涵盖线性方程、分配律、分数、指数法则、不等式和二次方程（例如，\(x^2 = 9\) 分支为 \(x = \pm 3\)）。
- **仅追加的推导树：** 支持撤销、废弃分支和情况分支。
- **布局几何：** 通过静态度量表实现（无需 DOM 或字体测量）。

**设计承诺：** 精确性、用于渲染的稳定 ID、经属性测试的有条件正确性，以及析取支持（分支规则产生多个活跃支路）。

**使用方法：** `parseEquation`、`Derivation`、`enumerateMoves`、`ruleById`、`exprToString` 和 `layoutNode` 函数。可在 Node、浏览器、Web Worker 和原生 WebView 中运行。该引擎采用 MIT 许可证；配套的 Wyrm Math 应用（iOS/Android）维持其持续开发。
