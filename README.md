## 6. 不要在通用元素（如div）上使用aria-label属性

**原文标题**: Don't put aria-label on generic elements like divs

**原文链接**: [https://www.matuzo.at/blog/2026/aria-label-generic-elements](https://www.matuzo.at/blog/2026/aria-label-generic-elements)

**摘要：避免在通用元素上使用 `aria-label`**

本文解释了为什么在 `<div>` 或 `<span>` 等通用元素上使用 `aria-label` 或 `aria-labelledby` 会对网页无障碍性造成问题。

**失败原因：**
1.  **ARIA 规范：** “通用”（generic）角色（div/span 的默认角色）被列为“无法命名”的角色。根据规范，从技术上讲，不允许对这些元素进行标记。
2.  **行为不一致：** 屏幕阅读器对已标记的通用元素处理方式大相径庭。测试表明，虽然 macOS 的 VoiceOver 和 Chrome 的 Talkback 会读取标签，但 JAWS、NVDA 和 iOS 的 VoiceOver 等其他工具会忽略作者定义的标签，转而播报元素的文本内容。这种不一致会给用户造成困惑。

**例外情况：**
-   **`<section>`：** 尽管其默认角色为“通用”，但添加 `aria-label` 会将其隐式角色更改为“区域”（region，一种地标），从而允许进行标记。
-   **`<div popover>`：** `popover` 属性会将角色更改为“组”（group），从而允许在不违反规则的情况下进行标记。

**关键要点：** 避免在纯 `<div>` 或 `<span>` 元素上使用 `aria-label`。如果需要提供无障碍名称，请使用语义化的 HTML 元素（如 `<section>`、`<nav>` 或 `<aside>`）或有意义的 ARIA 角色（如 `role="region"`）。

## 8. 研究发现：散步比静坐更能激发创造力（2014）

**原文标题**: Taking a walk may lead to more creativity than sitting, study finds (2014)

**原文链接**: [https://www.apa.org/news/press/releases/2014/04/creativity-walk](https://www.apa.org/news/press/releases/2014/04/creativity-walk)

**摘要：**

美国心理学会2014年发布的一项研究发现，与坐着相比，散步能显著提升创造性思维。研究人员进行了四项实验，共涉及176名参与者，其中大部分为大学生。在一项实验中，参与者在跑步机上行走、坐着或户外散步时完成创造力测试（例如，想出日常物品的其他用途）。结果显示，81%的参与者在行走时比坐着时能想出更多创意想法。这种效应并不仅限于活动进行时：刚结束散步的参与者即使在坐下后，其创造力水平仍持续偏高。散步类型（户外与室内跑步机）对结果影响不大，这表明提升认知的驱动力是散步这一身体行为本身，而不仅仅是环境。研究人员得出结论：散步能激发思想的自由流动，是一种简单而有效的提升创造力的方式，尤其适用于需要发散性思维的任务。但他们也指出，对于需要专注、聚合性思维的任务（例如解决只有一个答案的问题），散步的益处较小。该研究的主要作者玛丽莉·奥佩佐建议将散步融入日常生活，例如举行散步会议或在散步时进行头脑风暴，以激发创造潜能。

## 9. 加州在遭到反对后，采取措施将Linux从其年龄验证法中豁免。

**原文标题**: California moves to exempt Linux from its age-verification law after backlash

**原文链接**: [https://www.tomshardware.com/software/linux/california-moves-to-exempt-linux-from-its-upcoming-age-verification-law-after-backlash-over-forcing-operating-systems-to-collect-users-ages-amendment-proposed-by-the-same-lawmaker-who-wrote-the-original-law](https://www.tomshardware.com/software/linux/california-moves-to-exempt-linux-from-its-upcoming-age-verification-law-after-backlash-over-forcing-operating-systems-to-collect-users-ages-amendment-proposed-by-the-same-lawmaker-who-wrote-the-original-law)

**摘要**

加利福尼亚州正计划将Linux发行版从其颇具争议的年龄验证法——《加州年龄适当设计法案》（AB 2273）——中豁免，此前该法案遭到了强烈反对。这项原定于2024年生效的法律，要求在线平台估算或验证用户年龄以保护儿童。批评者指出，该法律措辞宽泛，可能迫使Linux等通用操作系统收集用户年龄，他们认为这对于开源软件而言既不切实际，也侵犯隐私。

为此，该法律的原起草人、众议员巴菲·威克斯提出了一项修正案（AB 2273修正案），明确豁免主要设计用于通用计算的操作系统，并特别点名Linux发行版。该修正案澄清，法律针对的是“主要面向儿童设计、营销或意图用于儿童”的平台，或未成年人使用率高的平台，而非开源操作系统。

此前反对声浪中包括对开源社区受损的担忧，开发者警告称，遵守该法律将需要对每位用户进行侵入性的年龄检查。此次豁免旨在维护Linux去中心化、尊重隐私的特性，同时仍允许该法律适用于与儿童互动的社交媒体或游戏等服务。该修正案目前正在立法程序中推进。
