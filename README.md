## 5. 不要轻信大上下文窗口

**原文标题**: Don't trust large context windows

**原文链接**: [https://garrit.xyz/posts/2026-05-06-dont-trust-large-context-windows](https://garrit.xyz/posts/2026-05-06-dont-trust-large-context-windows)

**总结：** 本文认为，大语言模型中的长上下文窗口很大程度上是一种营销噱头，因为模型存在一个“智能区域”（大约前10万个token）和一个超出此范围的“愚钝区域”，在该区域中注意力机制会退化，模型会遗忘较早的信息。尽管厂商宣称上下文窗口可达20万至200万个token，但研究证实有效上下文仅为宣传尺寸的一小部分。

这给编码智能体带来了问题，它们会因文件读取、调试会话和测试运行而迅速消耗token。像Claude Code这样的现代智能体会使用自动压缩（总结历史以重新开始），但这启动得太晚，且总结是由一个退化的模型生成的。

作者的解决方案是将上下文窗口视为一个预算，假设只有第一个区块是可靠的。他们通过手动编写的规范来开启新的会话，这提供了比自动摘要信噪比更高的交接。诸如 *obra/superpowers* 和 *mattpocock/skills* 等项目通过围绕小型、命名的工件（产品需求文档、计划、技能）构建工作流程来扩展这一思路，从而将信息从活跃会话中转移至工件中，供后续会话干净地读取。关键要点：通过有意将信息转移至书面工件，让工作会话保持在智能区域内。

## 9. 美国母乳样本中发现危险的内分泌干扰化学物质

**原文标题**: Dangerous hormone-disrupting chemicals found in US breast milk samples

**原文链接**: [https://www.theguardian.com/us-news/2026/jun/14/breast-milk-research-chemicals](https://www.theguardian.com/us-news/2026/jun/14/breast-milk-research-chemicals)

西雅图一项针对50位母亲母乳样本的研究发现，92%的样本含有包括双酚A、双酚S、三聚氰胺、氰尿酸、三氯生在内的有害激素干扰化学物质。这些内分泌干扰物会干扰对新生儿发育至关重要的激素，对婴儿发育构成重大风险。此前这些样本已被检测出含有全氟和多氟烷基物质（PFAS）"永久化学品"和阻燃剂。

主要作者、"无毒未来"组织的瑞安·巴贝迪指出，婴儿因处于快速发育阶段而尤其脆弱。尽管母乳喂养仍是最健康的选择，但研究结果凸显了由日用品（从个人护理用品到塑料制品）中广泛使用这些化学物质以及监管薄弱所导致的普遍系统性问题。

研究发现，74%和78%的样本中分别检测出与发育和生殖损害相关的双酚A和双酚S。62%的样本含有三氯生，92%含有三聚氰胺。虽然部分含量低于世界卫生组织可耐受摄入量标准，但先前研究表明这些物质仍可能致病。作者提醒，该研究样本量较小且参与者相对富裕，可能限制结果的普适性。

研究人员强调，个人消费者选择无法避免这些无处不在的化学物质，呼吁加强监管而非像当前环保署和国会正在推行的放松管制措施。
