## 2. 我发现1万个GitHub仓库在传播木马恶意软件

**原文标题**: I found 10k GitHub repositories distributing Trojan malware

**原文链接**: [https://orchidfiles.com/github-repositories-distributing-malware/](https://orchidfiles.com/github-repositories-distributing-malware/)

**摘要：**

一名安全研究人员发现超过1万个GitHub仓库正在传播特洛伊木马恶意软件。这些仓库并非分支，名称和贡献者各不相同，但有一个明显模式：它们克隆合法仓库，复制所有提交和贡献者以显得可信，然后反复推送名为"Update README.md"的提交，添加指向包含加载器与Lua DLL的恶意压缩包的链接。攻击者每隔几小时删除并重新推送同一提交，很可能为规避GitHub的检测算法。

研究人员利用GHArchive筛选每日更新频繁（1-24次）的仓库，再通过提交模式过滤。最初仅发现14个，调整过滤条件忽略"几小时"要求后，共找到1万个匹配仓库。每个仓库均托管一个被VirusTotal标记为木马的压缩包，尽管单个链接显示零检测结果。这些仓库已存在数月甚至超过一年，却未被GitHub自动检测系统识别。

研究人员推测该活动的目标包括：了解系统漏洞、通过伪造提交历史建立信任、以及利用低流量关键词和热门标签实现搜索引擎索引。结果公布后，GitHub最终开始删除这些仓库。同时，名为"Git Malware Finder"的检测脚本也已被发布。

## 6. 从 GNU Stow 迁移到 Chezmoi

**原文标题**: Migrating from GNU Stow to Chezmoi

**原文链接**: [https://rednafi.com/misc/chezmoi/](https://rednafi.com/misc/chezmoi/)

作者将GNU Stow迁移至Chezmoi，用于在三台Mac（工作机、个人机、服务器）及部分Linux虚拟机间管理点文件。Stow的符号链接模型导致仓库脏乱且多设备冲突，令其困扰不已。

**关键要点：**

- **Stow的局限：** 符号链接直接写入仓库，造成工作树混乱和冲突。Stow在引导时无法处理已存在的文件，且仅能管理文件，无法处理软件包或系统设置。

- **Chezmoi的方案：** 使用源目录（`~/.local/share/chezmoi`），采用常规文件（非符号链接）。`chezmoi add` 复制实际文件，编码后命名（如 `dot_zshrc`），`chezmoi apply` 再写回原位。模板（.tmpl）支持按设备配置数据；`private_` 等属性可设置权限。

- **引导流程：** 两条命令（`brew install chezmoi` + `chezmoi init --apply`）即可克隆仓库、通过Brewfile安装Homebrew包、部署点文件、运行macOS配置脚本——一步到位。

- **日常操作：** 通过 `chezmoi edit --apply` 编辑，或用 `chezmoi add` 同步实时变更；`chezmoi update` 在所有设备间同步。

- **Agent技能文件：** 将AI agent技能文件（Claude Code、Codex）统一存入 `~/.agents/skills`，并通过符号链接模板让Claude Code指向同一目录，确保两个agent共享文件且无重复。

作者的点文件仓库已公开，可供参考。

## 7. 医院和大学以更低成本重新利用药物

**原文标题**: Hospitals and universities repurposing drugs at lower cost

**原文链接**: [https://www.kcl.ac.uk/news/hospitals-and-universities-repurposing-drugs-at-90-lower-cost](https://www.kcl.ac.uk/news/hospitals-and-universities-repurposing-drugs-at-90-lower-cost)

**摘要：**  
伦敦国王学院一项发表于《剑桥法律杂志》的新研究揭示了医院和大学在药品专利体系之外运作的“隐形”药物再利用系统。这一平行体系以廉价仿制药开展晚期临床试验，成本比产业资助的试验低90%。  

关键发现：  
- 医院和大学资助的试验成本不到典型药企试验的10%，部分原因在于资源需求较低。  
- 再利用仿制药时，创新障碍（专业知识、风险、资本）显著降低，因为这些药物已获研究并已生产。  
- 动机与产业不同：临床医生和科学家关注患者康复及通过发表论文实现职业发展，而非专利驱动的利润。  
- 案例包括：用抗癌药治疗失明、将乳腺癌药物用于预防、用抗炎药治疗新冠。  

一旦药物成为仿制药，企业因竞争而丧失再利用兴趣，但这正是学术-医院系统活跃之时。各国政府开始正式认可并支持此类研究，其能提供平价的治疗方案。伦敦国王学院的约翰森·利迪科特博士指出，该系统具备持续为患者提供低成本疗法的潜力。
