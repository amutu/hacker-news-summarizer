## 1. 手写有益于大脑

**原文标题**: Writing by hand is good for your brain

**原文链接**: [https://nealstephenson.substack.com/p/writing-by-hand-is-good-for-your](https://nealstephenson.substack.com/p/writing-by-hand-is-good-for-your)

**摘要：** 本文提倡手写，尤其是使用钢笔，因为手写能通过解决问题和协调动作调动更多大脑区域，带来认知益处。作者指出，由于人工智能的兴起，手写考试重新流行，但许多学生缺乏正确的手写技能。

关于舒适、有效手写的关键实用建议：
- **避免使用铅笔和廉价圆珠笔**，因为它们需要过大力度，容易导致疲劳。
- **避免使用无摩擦力的工具**（如玻璃屏上的触控笔），因为少量摩擦力（“涩感”）有助于控制手部动作，减少脑力负担。
- **选用合适的笔纸组合**；廉价打印纸吸墨性差，而优质纸张（如25%棉含量纸张或空白笔记本）效果最佳。
- **不要节省纸张**——单面书写以保持自然风格，避免疲劳。
- **使用连笔字**，因为它比逐个书写字母更省力。
- **不必过度纠结字迹清晰或墨渍**；大多数书写为个人用途，现代钢笔极少造成问题。
- **左撇子写作者**可通过选择合适纸张和墨水，用钢笔书写而不弄脏纸面。
- **起初使用墨囊**以简化操作；先购买较便宜的钢笔。
- **百乐G-2中性笔**能以低成本提供类似钢笔的优势，但简单钢笔更耐用。

作者强调，即使用于列清单或记笔记，手写也能提升记忆力和思维活跃度，且只要使用恰当工具和技巧，不会导致抽筋。

**原文标题**: A solid-state “atomic channel” for separating rare earth elements

**原文链接**: [https://pme.uchicago.edu/news-events/news/cleaner-route-purifying-rare-earth-elements](https://pme.uchicago.edu/news-events/news/cleaner-route-purifying-rare-earth-elements)

**摘要：**芝加哥大学普利兹克分子工程学院（UChicago PME）与西北大学、阿贡国家实验室合作，开发出一种利用固态“原子通道”分离稀土元素的更清洁方法。该成果发表在《自然·化学工程》上，新工艺基于层状氧化锰，其内部通道宽度仅相当于几个水分子。当引入稀土离子混合物时，较重的镧系元素（水合壳较小）结合更紧密，而较轻的镧系元素（水合壳较大）则会撑开层间距，从而实现分离。

为进一步区分性质极为相似的元素，研究团队使用电流并添加镁离子作为“支架”来固定通道宽度。这使得几乎相同的稀土元素表现出截然不同的结合行为。例如，钕对镧的富集倍数从1.6倍提升至5.4倍，经过两次循环后纯度可达97%。该方法仅使用水和电力，避免了有毒有机溶剂和酸，相比当前工业提纯过程更加环保。虽然尚未达到大规模制造水平，但该技术为分离尺寸差异仅数埃的离子提供了全新设计思路。未来工作将测试该方法对更多镧系元素的适用性，并完善计算模型以更深入理解“钉扎效应”。
