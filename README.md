*   `DRAW` е’Ң `PLACE` еҗ‘еӣҫиЎЁж·»еҠ еӣҫеұӮпјҲдҫӢеҰӮзӮ№гҖҒзӣҙж–№еӣҫгҖҒж ҮжіЁпјүгҖӮ
*   `SCALE` е’Ң `LABEL` еӯҗеҸҘжҺ§еҲ¶зҫҺеӯҰиҰҒзҙ е’Ңж ҮйўҳгҖӮ

ејҖеҸ‘ ggsql зҡ„дё»иҰҒеҠЁжңәеҢ…жӢ¬пјҡ
1.  жңҚеҠЎдәҺд»Ҙ SQL дёәдёӯеҝғзҡ„еҲҶжһҗеёҲпјҢд»–д»¬еёҢжңӣиҺ·еҫ—еҸҜеӨҚзҺ°гҖҒеҹәдәҺд»Јз Ғзҡ„еҸҜи§ҶеҢ–пјҢиҖҢж— йңҖеҲҮжҚўеҲ° R/Python жҲ–дҫқиө– GUI е•ҶдёҡжҷәиғҪе·Ҙе…·гҖӮ
2.  еҲ©з”Ё SQL зҡ„еЈ°жҳҺејҸж•°жҚ®ж•ҙзҗҶдёҺеӣҫеҪўиҜӯжі•зҡ„еЈ°жҳҺејҸеҸҜи§ҶеҢ–д№Ӣй—ҙзҡ„еӨ©з„¶еҘ‘еҗҲжҖ§гҖӮ
3.  еҲӣе»әдёҖдёӘиҪ»йҮҸзә§гҖҒеҸҜеөҢе…Ҙзҡ„иҝҗиЎҢзҺҜеўғпјҢжҜ”е®Ңж•ҙзҡ„зј–зЁӢиҜӯиЁҖжӣҙе®үе…ЁгҖҒжӣҙй«ҳж•ҲпјҢиғҪеӨҹзӣҙжҺҘеңЁж•°жҚ®еә“еҗҺз«ҜдёҠжү§иЎҢд»ҘеӨ„зҗҶеӨ§еһӢж•°жҚ®йӣҶгҖӮ
4.  еҲ©з”ЁеӨ§еһӢиҜӯиЁҖжЁЎеһӢеҜ№ SQL зҡ„зІҫйҖҡиғҪеҠӣпјҢдёәиҮӘз„¶иҜӯиЁҖеҲ°еҸҜи§ҶеҢ–зҡ„жҷәиғҪдҪ“жҸҗдҫӣж”ҜжҢҒгҖӮ

иҜҘе·Ҙе…·дё“дёәз¬ҰеҗҲдәәдҪ“е·ҘзЁӢеӯҰгҖҒеҸҜиҝӯд»Јзҡ„еӣҫиЎЁжһ„е»әиҖҢи®ҫи®ЎпјҢзӣ®еүҚжҸҗдҫӣ alpha зүҲжң¬пјҢеҸҜеңЁ QuartoгҖҒJupyterгҖҒPositron е’Ң VS Code дёӯдҪҝз”ЁгҖӮ
