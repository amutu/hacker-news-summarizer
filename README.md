本文探讨了因办公椅产生的静电和电磁干扰（EMI）导致显示器间歇性黑屏或闪烁的问题。作者在使用连接MacBook的外接显示器时遇到了这一情况，尤其是在从宜家马库斯椅子上起身或移动时尤为明显。

根本原因被确定为椅子气压杆机制产生的EMI尖峰干扰了视频线缆——尤其是DisplayPort线缆和适配器。这是一个有文献记载的问题，DisplayLink的相关资料也佐证了这一点。

作者提出了两种解决方案：
1.  **将椅子接地**：为消散静电，可在椅子的金属框架上连接一条导电链至地面。这显著减少了触电现象。
2.  **使用磁环**：在视频线缆上夹装铁氧体磁环以抑制EMI尖峰。虽然这降低了显示器黑屏的频率，但未能完全消除问题，特别是在突然移动时。

结合使用椅子接地和磁环的方法有效缓解了该问题。

为破除这一迷思，他推荐了两项基础资源。首先，**Jack Crenshaw的《让我们构建编译器！》**教程系列因其简洁性受到赞誉，它展示了如何构建适合初学者的基础单遍编译器（类似早期的Turbo Pascal）。但文中也指出该方法缺乏内部抽象语法树（AST），限制了灵活性。

作者的核心建议是：先从这些注重实践而非理论的指南入手，后续如有需要再参考《龙书》这类厚重的理论著作。

## 8. Cal.com将转为闭源

**原文标题**: Cal.com is going closed source

**原文链接**: [https://cal.com/blog/cal-com-goes-closed-source-why](https://cal.com/blog/cal-com-goes-closed-source-why)

根据文章内容，以下是简明摘要：

流行的开源日程安排工具Cal.com宣布了其商业模式的重大转变。虽然其核心日程安排功能将继续在AGPLv3许可下保持开源，但该公司正将关键新功能及即将推出的“Cal.com平台”转向闭源专有模式。

这一变化的主要原因是经济可持续性。公司发现其先前“开放核心”模式（即通过托管服务销售高级功能）并不足够。大型企业和云服务提供商能够无偿使用功能齐全的开源代码，实质上与Cal.com自身的收入流形成竞争。这使得公司难以持续资助创新并进行规模化竞争。

主要变化包括：
*   **核心日程安排功能：** 保持开源（AGPLv3）。
*   **新高级功能：** 工作流、路由表单和分析等功能将成为新的闭源“Cal.com平台”的一部分。
*   **新许可协议：** 新的专有“Cal.com许可”将管辖新平台及其功能。
*   **对现有用户的承诺：** 当前自托管用户可以保留其部署版本，但不会获得高级/闭源组件的更新。新功能将仅通过官方Cal.com云服务或新的自托管企业版提供。

公司表示此举对于建立能够大力投资研发的持久业务是必要的，最终目标是打造强大的“互联网日程安排基础设施”。他们承认这可能会让开源社区的部分成员感到失望，但认为这对于公司的长期生存和创新能力至关重要。

**原文标题**: Good Sleep, Good Learning (2012)

**原文链接**: [https://super-memory.com/articles/sleep.htm](https://super-memory.com/articles/sleep.htm)

本文《优质睡眠、高效学习、美好生活》主张，高质量的自然睡眠是优化学习、创造力与健康的基石。文章综合睡眠科学知识提供实用建议，主要倡导**自由睡眠法**——即让身体在没有闹钟干扰的情况下自然入睡和醒来。

核心观点认为睡眠受两个过程调控：清醒时逐渐积累的稳态“压力”，以及控制警觉度的昼夜节律。良好睡眠需要使两者协调一致。作者强烈批评使用闹钟和强制睡眠时间表，认为这与失眠、学习效率低下以及轮班工作引发的健康问题密切相关。

主要建议包括：
*   根据个人需求优先保证睡眠的规律性与时长。
*   将**小睡**视为自然且有益于恢复认知活力的习惯。
*   利用睡眠日志等工具了解个人生理节律。
*   批评**多阶段睡眠法**对多数人而言不可持续且有害健康。
*   通过自由睡眠与精细的光照管理重新校准昼夜周期，以应对睡眠时相障碍（如睡眠时相延迟综合征）。

文章将睡眠生理机制与记忆巩固直接关联，指出睡眠对“神经优化”和学习至关重要。结论强调，尽管现代生活常与自然睡眠相冲突，但理解并尊重身体固有的睡眠机制，才是保持清醒活力与巅峰思维能力的可靠途径。
